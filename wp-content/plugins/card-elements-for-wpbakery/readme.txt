== Installation ==

**From Your WordPress Dashboard**

1. Go to Plugins > **Add New**.
2. Search for **Card Elements for WPBakery**.
3. Click on the **Install Now** button.
4. Click on **Activate Now** after installed in the backend.

**From WordPress.org** 

1. Download **[Card Elements for WPBakery](https://downloads.wordpress.org/plugin/card-elements-for-wpbakery.zip)**.
2. Upload the **'card-elements-for-wpbakery'** folder to the /wp-content/plugins/directory.
3. Activate **Card Elements for WPBakery** plugin from your plugins page.

== Frequently Asked Questions ==

= Can I add multiple card elements to a single page? =
Yes, you can. Card Elements for WPBakery plugin allows you to add **multiple card styles on the same page**. You need to drag and drop that module various times and select different card styles, whatever you want.

=  Can I change color or customize the card style as per my wish? =
Yes, you can customize it. We provided all possible options in the backend to update it from default card elements.

= Does Card Elements for WPBakery give me total control of my card styles? = 
Yes, Card Elements for WPBakery gives you total control of your card styles. You need to set up everything correctly and play with it. You may get more than your expectations.
